# AI 情绪健康与压力临床评估系统 (Medical v1.0)

本项目是一个基于面部微表情识别技术的**临床级心理健康监测与压力评估商业化原型**。它将前沿的深度学习技术与临床心理学标准相结合，为个人情绪管理、企业员工关怀及医疗辅助诊断提供数据支撑。

## 🌟 核心作用

1. **临床级压力监测**：
   - 基于 **PSS-10 (感知压力量表)** 标准，通过微表情频次与强度量化心理压力。
   - 模拟 **HRV (心率变异性)** 与 **瞬目频率 (Blink Rate)**，提供生理维度的严谨参考。
2. **心理健康预警**：
   - 实时识别 7 种核心情绪（开心、悲伤、愤怒、惊讶、恐惧、厌恶、平静）。
   - 自动生成风险等级（健康、中度预警、极高风险），并提供针对性的临床建议。
3. **商业化闭环流程**：
   - 包含完整的用户注册登录、会员订阅系统、充值密钥激活及 PDF 深度报告导出功能。
   - 符合 **HIPAA/GDPR** 数据隐私设计规范，所有操作均记录于审计日志 (Audit Logs)。

## 🚀 核心功能与用法

### 1. 环境准备
确保已安装 Python 3.10+，并运行以下命令安装依赖：
```powershell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 启动系统
在项目根目录下执行：
```powershell
$env:PYTHONPATH="."; python -m uvicorn backend.main:app --host 0.0.0.0 --port 8000 --reload
```
访问地址：[http://localhost:8000](http://localhost:8000)

### 3. 使用流程
1. **注册/登录**：创建个人账号以保存历史评估数据。
2. **系统校准 (Calibration)**：
   - 点击“基准校准”，保持面部平静 5 秒。
   - 系统将采集您的生理基准，确保后续评估的医疗严谨性。
3. **实时检测**：
   - 开启摄像头或上传视频/图片。
   - AI 引擎将分析微表情序列，并在仪表盘更新 PSS、HRV 等核心指标。
4. **生成临床报告**：
   - 点击“导出 PDF 报告”，获取包含数字化签名、唯一审计 ID 及专家建议的专业文件。
5. **会员升级**：
   - 通过充值中心输入密钥（演示密钥示例：`ME-PRO-TEST-2026`）激活高级权限。

## 🛠 技术架构
- **后端**：FastAPI (Python) + SQLite/MySQL + Redis。
- **AI 引擎**：支持 PyTorch 原生推理与 ONNX 高性能推理（具备 Demo 引擎优雅降级）。
- **前端**：原生 HTML5/CSS3 (玻璃拟态设计) + JavaScript (ES6)。
- **合规性**：内置 AES-256 加密模拟与全行为审计日志。
